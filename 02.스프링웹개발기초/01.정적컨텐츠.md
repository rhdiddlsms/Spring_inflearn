# 1. 정적 컨텐츠

파일을 웹브라우저에 그대로 보내주는역할

- [스프링 부트 정적 컨텐츠 기능 ](https://docs.spring.io/spring-boot/docs/2.3.1.RELEASE/reference/html/spring-bootfeatures.html#boot-features-spring-mvc-static-content)

- hello-static.html

```html
<!DOCTYPE HTML>
<html>
<head>
 <title>static content</title>
 <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
</head>
<body>
    이 파일은 정적 컨텐츠 파일입니다.
</body>
</html>
```

여기엔 프로그래밍을 할 수 없음. 그냥 있는 그대로 받아 들이게 된다.

## 1.1. 정적 컨텐츠 요약.

1. 웹브라우저에서 내장 톰켓서버로 `http://localhost:8080/hello-static.html` 전달.
   1. 내장 톰켓 서버에서 스프링으로 넘겨줌
      1. controller에서 찾아봄. 
      2. 없으면 아래 항목으로
   2. resources: static/hello-static.html의 파일이 있나 찾아봄
2. 찾으면 웹브라우저로 hello-static.html을 보여줌.
