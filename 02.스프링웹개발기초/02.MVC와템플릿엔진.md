# 1. MVC

MVC: Model, View, Controller

- Model, Controlloer : 비즈니스 로직과 관련이 있거나 처리하는데 집중
- View : 화면을 그리는데 집중

## MVC, template 기본 틀

Controller 
```java
@Controller
public class HelloController {
    @GetMapping("hello-mvc")
    public String helloMvc(@RequestParam("name") String name, Model model) {

    model.addAttribute("name", name);
    return "hello-template";

    }
}
```


hello-template.html
```html
<html xmlns:th="http://www.thymeleaf.org">
<body>
<p th:text="'hello ' + ${name}">hello! empty</p>
</body>
</html>
```

### 1.1.1. 실행

`http://localhost:8080/hello-mvc?name=spring`

주소창에 복사 붙혀넣기.

## MVC, 템플릿 엔진 진행방식

1. 웹브라우저에서 hello-mvc로 맵핑된 주소를 내장 톰켓 서버로 전달
2. 내장 톰켓 서버에서 스프링 컨테이너인 helloController로 전달.
   1. return : hello-template
   2. model(name:spring)
      1. key는 name 값은 spring
3. ViewResolver 가 받아서 template/hello-template.html(Thymeleaf 템플릿 엔진 처리) 를 HTML 변환 후 웹브라우저로 전달.
