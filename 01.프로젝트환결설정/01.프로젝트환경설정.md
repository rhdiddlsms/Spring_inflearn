# 1. 시작

- [(start.spring.io) 스프링 프로젝트 초기설정파일 만들기 사이트 ](https://start.spring.io/)

![사이트 접속 후 설정방법(이미지)](/img/1강프로젝트초기설정.PNG)

1. Group : 패키지명
2. Artifact : 프로젝트명
3. Dependencies 클릭 후 Spring Web 추가하기.
4. Dependencies 클릭 후 Thymeleaf 추가하기.
5. GENERATE 눌러서 파일저장하기.
6. 자신의 workspace에 압축풀기.
7. open & import를 눌러서 build.gradle 파일 open하기
8. ✅ 된 것들 알맞게 체크하기.

  - Language: Java

  - Java: 11

# 2. 프로젝트 실행

- src
  - main
    - java : 내가 사용할 코드
    - resources : xml, html, 설정파일 등 java파일을 제외한 다른파일들.
  - test
    - test코드와 관련된 파일들.
    - 요즘 개발 트랜드에서는 중요.
- build.gradle ✔
  - 깊게 공부는 아직은 괜찮고, 버전설정, 라이브러리 가져오기 정도로만 알고있자.
  - sourceCompatibility : java버전
  - repositories {	mavenCentral() }
    - mavenCentral 이 사이트에서 다운로드를 받으라고 프로그램에 말하는 용도.
  - dependencies { 내가 선택한 라이브러리 가 있음. }
- .gitignore
  - start.spring.io에서 만들어줌.
